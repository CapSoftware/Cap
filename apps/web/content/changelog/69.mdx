- New scenes feature, allowing you to make the camera fullscreen for sections of the video, or hide it
- Higher quality GIF exports
- Custom cursor capture is now enabled by default (can be disabled in Settings > Experimental)
- New camera preview is more stable
- Setting a background will now automatically add padding if there is none
- New recording flow (can be enabled in Settings > Experimental)
- New recording engine behind the scenes, allowing us a better platform to build on for future features
