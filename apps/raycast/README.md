# Cap Raycast Extension (deeplink-based)

This Raycast extension controls Cap Desktop through `cap-desktop://` deeplinks.

## Commands

- Start Cap Recording
- Stop Cap Recording
- Toggle Pause Cap Recording
- Switch Cap Microphone
- Switch Cap Camera

## Development

```bash
cd apps/raycast
pnpm install
pnpm dev
```

Cap Desktop must be installed and running (or launchable) so deeplinks can be handled.
